# AURA: Authenticity Scan 🔍

A futuristic web application that analyzes videos to detect AI-generated content and deepfakes.

## ✨ Features
- 🎥 Video upload (drag & drop or click)
- 🔬 AI deepfake detection simulation
- 📊 Beautiful animated UI
- 🌐 URL pasting support

## 🛠️ Technologies
- HTML5
- CSS3 (Animations, Glassmorphism)
- JavaScript (ES6+)

## 🎮 Demo
Press **'D' key** to toggle between FAKE/REAL results.

## 📂 Files
- `index.html` - Main HTML file
- `styles.css` - All styling
- `script.js` - All JavaScript functionality
